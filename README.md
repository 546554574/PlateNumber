# PlateNumber
车牌选择弹框

使用方法：
第一步：
gradle版本
Add it in your root build.gradle at the end of repositories:

```aidl
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

	
maven版本

```aidl
<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
```
	
第二步：
```aidl
dependencies {
	        implementation 'com.github.546554574:PlateNumber:v1.1'
	}
```
